# Slide-Up Popover – Product Catalog A responsive and accessible **pure CSS Slide-Up Popover** designed for product catalog interfaces. ## Features - Pure CSS (No JavaScript) - Smooth Slide-Up animation - Responsive layout - Keyboard accessible - Uses CSS custom properties - Product catalog themed design ## Files ``` demo.html style.css README.md ``` ## Customization ```css :root{ --ease-popover-duration:0.35s; --ease-popover-easing:ease; --ease-popover-distance:16px; } ``` ## Accessibility - Keyboard focus support - `role="tooltip"` - Visible focus outline - Responsive on mobile devices ## Usage Hover or focus the button to display the Slide-Up popover.
